## [1Week] 박진영

### 미션 요구사항 분석 & 체크리스트

---  

**[필수과제]** 호감 상대 삭제하기
- [ ] 호감목록 페이지에서 특정항목 삭제버튼 클릭 시, 해당 항목 삭제
    - [ ] 삭제 전, 해당 항목에 대한 소유권이 로그인한 사람에게 있는지 확인
- [ ] 삭제 후, 호감목록 페이지로 돌아오기

**[선택과제]** 구글 로그인 기능 구현
- [ ] 구글 API 가져오기
- [ ] 구글을 통한 회원가입
- [ ] 구글을 통한 로그인 처리
